# ModernMart – Next-Gen Shopping Experience

ModernMart is a modern e-commerce platform designed to provide a seamless and interactive shopping experience. Built with **Next.js**, **Firebase**, **Node.js**, **Express**, and **MongoDB**, it combines modern frontend and backend technologies for a full-stack solution.

## Live Demo

[ModernMart Live](https://modern-mart-eta.vercel.app/)

## Features

* **Show Products** – Browse all available products easily.
* **Create Product** – Admins can add new products.
* **Manage Products** – Update or delete existing products.
* **Private Routes** – Secure pages requiring authentication.
* **Responsive Design** – Works on both desktop and mobile devices.

## Tech Stack

* **Frontend:** Next.js, React, Tailwind CSS
* **Backend:** Node.js, Express
* **Database:** MongoDB
* **Authentication & Hosting:** Firebase
* **Deployment:** Vercel
